# NinjaScript-Tools

Collection of custom NinjaScript tools developed for portfolio management, automated trading, and quick order entry.

## Folder Structure

- `/AddOns/` - Custom AddOns (e.g. QuickMouseEntryATM)
- `/Indicators/` - Custom indicators
- `/Strategies/` - Trading strategies
- `/Utils/` - Helper classes and shared code

## Installation

1. Open NinjaTrader 8
2. Go to **Tools → Edit NinjaScript** → Right-click and **New AddOn** / **New Indicator** etc.
3. Copy the .cs file content into the editor
4. Compile (F5)
5. Add the AddOn/Indicator to your chart

## Current Tools

- **QuickMouseEntryATM** - Mouse price entry with ATM strategy using Alt+B / Alt+S hotkeys
